# AI Resume Screening System

## 🚀 Project Overview
AI Resume Screening System is an NLP-based application that analyzes resumes and compares them with a Job Description using TF-IDF and Cosine Similarity.

This system helps recruiters quickly identify the best candidates by calculating ATS match scores.

---

## ✨ Features
- Upload multiple PDF resumes
- Job Description matching
- ATS Score calculation
- Candidate ranking system
- Skills found & missing skills detection
- Clean Streamlit UI dashboard

---

## 🧠 Technologies Used
- Python
- NLP (TF-IDF)
- Scikit-learn
- Streamlit
- PDFPlumber

---

## 📊 Working Flow
1. Upload resumes
2. Paste Job Description
3. Text extraction from PDFs
4. TF-IDF vectorization
5. Cosine similarity scoring
6. Candidate ranking
